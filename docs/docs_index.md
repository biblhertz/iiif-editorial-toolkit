## 🔍 Quick Reference

### Essential Tools
- **Enhanced Academic Generator**: Complex comparisons, dual output
- **OpenSeadragon Generator**: Simple layouts, OSD-optimized
- **OpenSeadragon Viewer**: High-performance viewing
- **Multi-Viewer Testing**: Compatibility validation

### Key Workflows
- **Individual → Library → Comparison**: Standard academic workflow
- **Auto-fetch → Generate → Validate**: Quality assurance workflow
- **Create → Test → Deploy**: Production workflow

### Layout Types
- **Horizontal**: Basic side-by-side
- **Horizontal Balanced**: Equal heights for comparison
- **Storyboard**: Optimized viewing experience
- **Main + Derivatives**: Hierarchical display
- **Grid 2x2**: Systematic arrangement
- **Vertical**: Sequential stacking

### Viewer Compatibility
- **Universal (Single Canvas)**: Mirador 3, TheseusViewer
- **Simple (Multi-Canvas)**: Basic IIIF viewers
- **Dual Output**: Maximum compatibility
